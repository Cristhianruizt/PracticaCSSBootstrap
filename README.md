# Proyecto: Proyección Profesional - Integración Bootstrap 5

## 📝 Descripción del Proyecto
Este proyecto representa una nueva etapa en el proceso de aprendizaje de **Fundamentos Web**. Tras haber trabajado la estructura con HTML y la presentación con CSS, esta práctica se enfoca en la migración y desarrollo de una interfaz moderna y responsiva utilizando el framework **Bootstrap 5**.

El sitio se centra en la **proyección personal, académica y profesional**, detallando metas a futuro, habilidades por fortalecer y la visión de carrera del estudiante hacia el año 2026.

## 📂 Estructura de Carpetas
La organización del proyecto sigue una jerarquía lógica para el mantenimiento de recursos:

- `PracticaBootstrap/`
  - `index.html` (Página de inicio adaptada con Bootstrap)
  - `pages/`
    - `proyeccion.html` (Página principal de la práctica)
    - `buscar.html` (Búsqueda de información)
    - `cto/contacto.html` (Formulario de contacto)
  - `css/`
    - `general.css` (Estilos base)
    - `proyeccion.css` (Personalización específica de Bootstrap)
  - `img/` (Logos e imágenes institucionales)
  - `video/` (Contenido multimedia local)
  - `audio/` (Recursos de audio local)

## 🚀 Tecnologías Utilizadas
* **HTML5:** Uso de etiquetas semánticas (`header`, `nav`, `main`, `section`, `article`, `figure`, `footer`).
* **Bootstrap 5.3.3:** Framework principal para el diseño responsivo y componentes.
* **CSS3:** Ajustes visuales personalizados y sobreescritura de estilos del framework.
* **Mobile First:** Enfoque de diseño prioritario para dispositivos móviles.

## 🛠️ Componentes Bootstrap Implementados
Siguiendo los requerimientos de la rúbrica, se han integrado y combinado los siguientes componentes:

1. **Navbar:** Menú de navegación responsivo con colapso automático para móviles.
2. **Grid System:** Uso de contenedores, filas y columnas para una distribución adaptativa.
3. **Cards:** Organización de habilidades y metas con bordes y sombras.
4. **Badges:** Etiquetas para resaltar el estado de las metas (Prioridad, En curso, Meta).
5. **Progress Bars:** Visualización de niveles de dominio dentro de una tabla.
6. **Accordion + Carousel:** Un acordeón informativo que contiene en su interior un carrusel de proyectos futuros.
7. **Modal + Forms:** Ventana emergente interactiva que contiene un formulario de contacto.
8. **Utilities:** Aplicación de clases de espaciado, tipografía y sombras.

## 🎨 Personalización
Se ha implementado el archivo `proyeccion.css` para personalizar la apariencia de Bootstrap, ajustando:
- Colores institucionales (Verde y Dorado).
- Efectos de transición en tarjetas (`hover`).
- Estilos específicos para el encabezado y pie de página.
- Bordes personalizados en secciones semánticas.

## 📱 Enfoque Mobile First
La página ha sido verificada para funcionar correctamente en:
- **Móvil:** Menú tipo hamburguesa, tarjetas apiladas verticalmente y tablas con scroll horizontal.
- **Tablet/Escritorio:** Rejilla de 3 columnas para tarjetas y navegación extendida.
